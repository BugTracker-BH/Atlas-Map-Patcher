## Overview
The **Atlas Tool Suite** is a unified, highly optimized single-file web application specifically engineered for Minecraft map creators. Submitting a world to Atlas has strict technical requirements—oftentimes, third-party parsers crash due to corrupted `0-byte` region files, and specific promotional dimension rules must be adhered to.

This lightweight tool operates **100% inside your browser**. Your map stays securely entirely on your machine.

---

## Module 1: Atlas Map Patcher
Minecraft `.mca` region files technically require an 8192-byte header. However, Minecraft casually ignores `0-byte` or completely empty region chunks generated by world editors. Unfortuantely, when uploading a map, server-side tools (like Node.js Buffers on Atlas) violently crash with `RangeError [ERR_BUFFER_OUT_OF_BOUNDS]` upon seeking these corrupted maps.

The **Map Patcher** aggressively seeks and destroys these invalid `.mca` elements in bulk. 

### How it works:
1. Validates the incoming `.zip` structure.
2. Isolates and sequentially analyzes every `.mca` binary via memory buffering.
3. Eradicates all instances `< 8192` bytes without touching valid chunks to retain maximum fidelity.
4. Generates an instant, highly optimized patched `.zip` ready for flawless submission.

## Module 2: Promo Assets Converter
Atlas specifically dictates: *4-10 screenshots for internal use, must be `1280x720` and in `.jpg`.*

The **Promo Converter** takes any image size, orientation, or file type generated by your screenshot tool, and intelligently crops, scales, and reformats them without stretching.

### Feature Mechanics:
* **Bulk Processing:** Drop 20 screenshots at once, and it converts them all flawlessly.
* **Canvas `object-fit: cover` Logic:** It actively measures aspect ratios. If an image is too wide or too tall, it precisely scales and center-crops the image to perfect `1280x720` pixels.
* **Format Forcing:** Guarantees all outputs are exactly `.jpg` using `0.95` high-quality JPEG compression settings.
* **Immediate Zipping:** To avoid browser spam blockades, it zips all processed promotional assets straight into `Atlas_Promo_Assets_1280x720.zip` ready to be attached to your ticket.

## How to Use
1. Open the raw `atlas_map_patcher.html` logic tool in any modern browser (Chrome, Edge, Firefox).
2. Toggle between **"Map Patcher"** and **"Promo Converter"** using the custom chiclet tab system in the bottom left.
3. Drag and Drop your `.zip` world, or a bulk selection of raw images into the specific workspace.
4. View the embedded live-log terminal. 
5. Process directly onto your drive.
